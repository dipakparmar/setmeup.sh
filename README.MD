# setmeup.sh

a tiny library to set your things up :wink: smartly with one command or interactively. 

## Installation

No Installation required in the current version! 

```bash
sh ./setup.sh
```

## Usage

```bash
Do you wish to install Browsers x y z ?
Enter with y or n
```
